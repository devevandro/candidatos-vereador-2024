## Detalhes do Projeto
- Projeto criado com intuito de mostrar os candidatos a vereador da cidaee de Cornélio Procópio

## Como compilar
com NPM
````
npm run dev
````
YARN
````
yarn dev
````
Estará disponível na porta 3000
http://localhost:3000
